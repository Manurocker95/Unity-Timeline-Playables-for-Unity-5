# Unity-Timeline-Playables-for-Unity-5

Backport of Unity's Playables (Timeline) for Unity 5. It is ment to be used on Unity 5.6.6f2 for Nintendo 3DS.

The code was decompiled from Unity's 2017.1 + 5.6.0beta DLL with DNSpy and backported to Unity 5. It works on Editor and Runtime, but keep in mind that the Timeline Window is NOT present in this project. However, you can use [SLATE](https://assetstore.unity.com/packages/tools/animation/slate-cinematic-sequencer-56558) as the timeline editor with this package. 

In order to check if a track works, I added a simple example to test audio and animation track.

Feel free to make the timeline editor window and make a PR.
